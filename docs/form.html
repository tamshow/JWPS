form-layout

【フォームの基本形】
.e-form-table = 大枠
.e-form-unit = 1かたまり
.e-form-head = 見だし
.e-form-field = フィールド
※.e-form-fieldの中は.e-form-cols > .e-form-col12 等で囲みその中にinputタグを記述

【レイアウト】
.e-form-cols = グリッドレイアウト枠
.e-form-col[0_5 - 12] = グリッドレイアウト スマホ時は続けて .e-form-col12--sm等を記述

【ヘルパー】
.e-form-control = input, textarea, selectなどのinput要素
.e-form-text-dash = .e-form-field内の「 - 」を囲むタグ.e-form-col0_5の中に記述
.e-form-inframe = .e-form-field内インラインフレーム
[role="tooltip"] = 入力タイプ説明をツールチップで紐付ける
.e-form-req = 必須
.e-form-any = 任意
.e-form-text = フォームテキスト
.e-form-small-text = フォームスモールテキスト
